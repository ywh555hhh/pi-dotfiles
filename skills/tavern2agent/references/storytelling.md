# 叙事节拍

可选。GM 卡壳时读，不要整篇塞 prompt。

在 `gm.md` 里引用：

```md
叙事卡壳时，参考 references/storytelling.md 的场景三问和五步循环。
```

## 场景三问

1. 本场什么价值变了？
2. 从什么状态变成什么状态？
3. 没有变化，为什么存在？

每轮至少推动一个微变化：信任→怀疑，安全→危险，亲近→疏离。

## 三个节奏引擎

### 信息不对称

| 谁知道更多 | 效果 |
|---|---|
| 玩家 > 角色 | 戏剧性反讽 |
| 角色 > 玩家 | 神秘感 |
| 双方都不全知 | 悬念 |

真相分层揭示。每层回答旧问题，同时抛新问题。

### 打破预期

熟悉原型 + 意外变化：反类型角色、陌生背景、类型混合、动机反转。

### 情感波浪

推高 → 拉低 → 再推高。连续 3 轮平稳就制造扰动；高潮后留 1-2 轮余韵。

## 长线锚点

不用完整规划，但可记三个节点：

- 中点：局势或目标翻转。
- 一无所有：主角最低点。
- 灵魂黑夜：重新选择代价。

## RP 五步循环

1. 触发：NPC、环境、骰子结果推动行动。
2. 递进：冲突升级，赌注提高。
3. 逆转：每 3-5 轮给一次意外。
4. 高潮：情绪/冲突释放。
5. 余韵：短暂喘息，再开新循环。
